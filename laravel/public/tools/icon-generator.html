1000 A.D. Icon Prompt Generator

Copy these prompts into DALL-E, Midjourney, or any AI image generator to create consistent medieval game icons.

Style Guide (applies to ALL icons)

Art style: Hand-painted medieval game icon, reminiscent of Age of Empires II

Colors: Warm golds, ambers, deep browns. Matches the game's dark fantasy palette.

Lighting: Soft top-left light source, subtle golden rim glow

Perspective: Front-facing, flat/isometric. Not 3D rendered.

Size: Generate at 1024x1024, will be displayed at 48x48 (desktop) and 32x32 (mobile)

Background: Transparent (or solid dark #1a1510 if your tool doesn't support transparency)

CRITICAL — Small-size readability: Icons MUST be identifiable at 32x32 pixels. Use ONE bold central object that fills 70-80% of the frame. Strong silhouette with thick outlines. High contrast between subject and background. NO fine details, NO tiny text, NO busy backgrounds. Each icon should be distinguishable from all others by shape alone. Think "bold logo" not "detailed illustration".

Color coding: Each icon should have a dominant color that differs from similar icons (e.g. gold mine = gold tones, iron mine = blue-grey tones).

Batch Generation Tip

For best consistency, generate all icons in a single session. If your tool supports style references, generate the first icon, then use it as a style reference for the rest.

Test at target size: After generating, shrink each icon to 32x32 in your image editor. If you can't tell what it is at a glance, regenerate with a simpler composition.

Save each icon as a PNG to: public/images/icons/buildings/[name].png or public/images/icons/soldiers/[name].png
