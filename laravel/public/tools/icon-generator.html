1000 A.D. Icon Prompt Generator

Copy these prompts into DALL-E, Midjourney, or any AI image generator to create consistent medieval game icons.

Style Guide (applies to ALL icons)

Art style: Flat heraldic emblem / shield icon. Bold, graphic, simplified. Like a coat of arms symbol — NOT a detailed painting or illustration.

Colors: Each icon gets ONE bright dominant color against a very dark background (#1a1510). Warm golds, bright reds, vivid blues, strong greens — saturated and punchy, not muted.

Lighting: Flat with a subtle golden edge glow. No complex shadows or atmospheric lighting.

Perspective: Flat front-facing. No perspective, no 3D, no depth.

Size: Generate at 1024x1024, will be displayed at 48x48 (desktop) and 32x32 (mobile)

Background: Solid dark #1a1510 or transparent. Completely clean — no ground, no sky, no scenery.

CRITICAL — 32px readability: These icons MUST be instantly recognizable at 32x32 pixels. Rules:

  • ONE object only. A single bold symbol — not a scene, not a building, not multiple items.
  • Fills 85-90% of the frame. The object should be BIG. Almost touching the edges.
  • Thick outlines. Think 3-4px outlines at 1024px. Crisp edges, not painterly/blurry.
  • Flat color fills. 2-3 colors max per icon. Large solid color areas, not gradients or textures.
  • No interior detail. No wood grain, no stone texture, no fabric folds. Just solid shapes.
  • Unique silhouette. Each icon must have a completely different outline shape from every other icon.

Think: Mobile app icons, emoji, road signs — NOT oil paintings or game splash art.

Batch Generation Tip

For best consistency, generate all icons in a single session. If your tool supports style references, generate the first icon, then use it as a style reference for the rest.

Test at target size: After generating, shrink each icon to 32x32 in your image editor. If you can't tell what it is at a glance, regenerate with a simpler composition.

Save each icon as a PNG to: public/images/icons/buildings/[name].png or public/images/icons/soldiers/[name].png
